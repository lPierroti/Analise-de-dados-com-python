# Analise-de-dados-com-python
Projeto de análise de dados em Python para identificação de padrões de cancelamento de clientes. Utilização de bibliotecas como Pandas e Plotly para tratamento de dados, análise exploratória e visualização gráfica, buscando compreender fatores associados à evasão de clientes e apoiar tomadas de decisão orientadas por dados.
